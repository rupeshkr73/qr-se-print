🔑 Razorpay account + keys kaise milenge (step by step):
  1. Account banao: dashboard.razorpay.com/signup
  2. Business details + KYC bharo. Website/App URL mein likho: qrseprint.in
  3. Account activate hone ka wait karo (KYC approve — 1-3 din)
  4. Phir Dashboard → Account & Settings → API Keys → Generate Live Key
  5. Key ID (rzp_live_ se shuru) upar paste karo
  6. Key Secret sirf EK BAAR dikhta hai — turant copy karke upar paste karo
⚠️ rzp_test_ wali (Test) keys mat daalna — usse asli paisa nahi aayega. Sirf rzp_live_ wali.
🔑 PhonePe merchant account + keys kaise milenge (step by step):
  1. PhonePe Business account banao: business.phonepe.com
  2. Register karke "Payment Gateway" (website ke liye) activate karne ki request karo
  3. Business details mein Website URL likho: qrseprint.in
  4. Approval ke baad Business dashboard ke Developer / API Keys section mein milenge: Merchant ID (MID), Salt Key, Salt Index
  5. Teeno upar paste karo — Salt Index zyada tar 1 hota hai
⏳ PhonePe PG approval mein 2-7 din lag sakte hain. Jaldi shuru karna ho to Razorpay use karo ya Sirf Counter Cash se start karke baad mein Shop Login → Settings se gateway add karo.
💡 Keys abhi nahi hain? Upar "Sirf Counter Cash" select karo — baad mein Shop Login ke settings se gateway add kar sakte ho.
