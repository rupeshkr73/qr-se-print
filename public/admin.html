📖 Keys Kaise Milegi? (Step-by-Step) ▼
  1. Niche button se Razorpay Dashboard kholo (agar account nahi hai, "Sign Up" karo)
  2. Dashboard khulne par left menu mein "Account & Settings" pe click karo
  3. Usme "API Keys" option dhundo aur click karo
  4. Upar right corner mein "Live Mode" select karo
  5. "Generate Key" button dabao
  6. Mobile pe OTP aayega, daal do
  7. Ek popup khulega jisme Key Id aur Key Secret dikhega — turant copy kar lo, Key Secret dobara nahi dikhega!
  8. Dono yahan paste kar do neeche
🔗 Razorpay Dashboard Kholo

⚠️ Pehli baar Live keys banane se pehle Razorpay aapki shop/business details verify kar sakta hai (1-3 din lag sakte hain). Tab tak "Sirf Counter Payment" mode use kar sakte ho.

📖 Keys Kaise Milegi? (Step-by-Step) ▼

  3. Left menu mein "Developer Settings" ya "API & Webhooks" dhundo
  4. Merchant ID aur Salt Key wahan milegi
  5. Dono yahan paste kar do neeche
