📖 Keys Kaise Milegi? (Step-by-Step) ▼
  1. Business/Website URL puchhe to likho: qrseprint.in
  2. Niche button se Razorpay Dashboard kholo (agar account nahi hai, "Sign Up" karo)
  3. Dashboard khulne par left menu mein "Account & Settings" pe click karo
  4. Usme "API Keys" option dhundo aur click karo
  5. Upar right corner mein "Live Mode" select karo
  6. "Generate Key" button dabao
  7. Mobile pe OTP aayega, daal do
  8. Ek popup khulega jisme Key Id aur Key Secret dikhega — turant copy kar lo, Key Secret dobara nahi dikhega!
  9. Dono yahan paste kar do neeche
🔗 Razorpay Dashboard Kholo

⚠️ Pehli baar Live keys banane se pehle Razorpay aapki shop/business details verify kar sakta hai (1-3 din lag sakte hain). Tab tak "Sirf Counter Payment" mode use kar sakte ho.
