📌 Zaroori Instructions

1️⃣ Hamesha latest version install rakho — upar wale green button se. Naye features (duplex, counter approval) aur bug fixes latest version mein hi milte hain.
2️⃣ Software usi PC par install karo jisse printer juda hai aur jo shop time par chalu rehta hai.
3️⃣ Install ke baad apna Shop ID daalo — bas, ho gaya. Python ya SumatraPDF alag se install karne ki koi zaroorat nahi, sab software ke andar bundled hai.
4️⃣ Naya version install karne se pehle purana uninstall karne ki zaroorat nahi — naya khud replace kar deta hai.
5️⃣ Agent tray icon (neeche right corner) mein 🖨️ dikhta rahe = sab theek. Wahan se "Check for Update" bhi kar sakte ho.
